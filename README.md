# 🚀 Sanjai Sankar - MERN Stack Developer Portfolio

Welcome to the official repository for my personal developer portfolio! This modern, interactive web application showcases my full-stack projects, core technical skills, experience, and service packages as a **MERN Stack Developer & Digital Marketer**.

---

## ✨ Features

- 🌓 **Dynamic Theme Switcher**: Instant Light & Dark mode toggling with persisted user preferences.
- 🎨 **Modern Visual Aesthetics**: Glassmorphism UI, ambient glow effects, responsive grid, and fluid micro-animations.
- 📄 **One-Click Resume Download**: Direct download link to Sanjai Sankar's professional PDF resume.
- 💼 **Interactive Project Showcase**: Dynamic filtering of web development projects.
- 📱 **Fully Responsive Layout**: Optimized for mobile, tablet, desktop, and large displays.
- 📬 **Contact Form & Service Packages**: Structured deal packages and easy-to-use contact section.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Properties, Flexbox, Grid, Keyframe Animations), Modern JavaScript (ES6+)
- **Design System**: Typography via Google Fonts (Inter & Outfit), Custom SVG Icons
- **PDF Resume**: Tailored PDF integration for resume downloads
